## Cuerpo del mensaje
Ver [`acpasion-bricolaje.bbcode`](acpasion-bricolaje.bbcode) — en formato **BBCode**
(phpBB), listo para copiar y pegar.

## Notas
- **Imágenes**: el post enlaza a las capturas en GitHub (`raw.githubusercontent.com`).
  Muchos foros las muestran sin problema, pero si acpasión bloquea el hotlink,
  subir como **adjuntos** las de [`../img/`](../img/) (`01_overview.png`,
  `09_ajustes.png`, `10_ajustes_frigo.png`, ya optimizadas 32-67 KB) y sustituir
  cada `[img]...[/img]` por el adjunto que genere el foro.
- **Formato**: BBCode. Si el editor del foro fuera Markdown o WYSIWYG, hay que adaptarlo.
- **Tono**: divulgativo, sin tecnicismos; solo lo que hace y que es funcional.
